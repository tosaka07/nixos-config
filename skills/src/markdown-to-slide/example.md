# Welcome to Marp
**Creating Presentations with Markdown**

---

## What is Marp?

- Markdown Presentation Ecosystem
- Convert Markdown to beautiful slides
- Supports PDF, PPTX, and HTML output
- Simple syntax, powerful results

---

## Key Features

1. **Easy to write** - Just use Markdown
2. **Flexible themes** - Built-in and custom
3. **Export options** - PDF, PPTX, HTML
4. **Live preview** - Watch mode for instant feedback
